Provenance Data for the RAVE project

Details about RAVE are available here: RAVE webpage.

This is a simple webapp for displaying provenance information of RAVE data, using the Django framework. It uses the data model currently discussed in the IVOA, which is based on the W3C ProvDM data model.

Pick one of the classes below to show the available records.

  • Activity
  • Entity
  • Agent

For one activity, the xml or provjson representation is shown by appending /xml or /json to the existing detail url, e.g.:

  • details: /provapp/activities/rave:act_dataextraction/
  • xml: /provapp/activities/rave:act_dataextraction/xml/
  • json: /provapp/activities/rave:act_dataextraction/json/

Choose one of the following links to explore more provenance graphs:

  • Detailed provenance: Form for RAVE_Obs_Ids
  • Overview: Graph of the main workflow
  • Overview: Provn representation
