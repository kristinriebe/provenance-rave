RAVE Provenance

Provenance webapp prototype for the RAVE project

Details about the RAVE project are available here: RAVE webpage.

This is a prototype webapp for displaying provenance information of RAVE data, using the Django framework and the provenance package django-prov_vo.

Provenance specific links etc. can be found via the following link:
