RAVE Provenance

Provenance webapp prototype for the RAVE project

Details about the RAVE project are available here: RAVE webpage.

This is a prototype webapp for displaying provenance information of RAVE data, using the Django framework. It uses the data model currently discussed in the IVOA, which is based on the W3C ProvDM data model.

Pick one of the classes below to show the available data using the REST api.

  • Activity
  • Entity
  • Agent

Append the id of a specific record to get its details. You can also add new records and update or delete existing ones.

  • details: {% url 'provapp:activity-detail' 'rave:act_dataextraction' %}

For one activity, the json representation is shown by appending ?format=json to the existing detail url, e.g.:

  • json: /provapp/activities/rave:act_dataextraction/?format=json/
xml and provn serialisations still need to be defined, e.g. like this ?format=xml

Forms:

  • Detailed provenance: Form for RAVE_Obs_Ids

Complete provenance (overview):

  • Overview: Graph of the main workflow
  • Overview: Provn representation
  • Overview: Pretty Provn representation (with clickable links)
